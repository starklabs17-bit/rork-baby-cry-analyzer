# Baby Cry Analyzer — Soothing AI-Powered Cry Detection

## Features

- **Record & Analyze in Real-Time** — Tap a button to start listening; the app continuously monitors your baby's crying and provides a live analysis of the likely reason
- **AI-Powered Cry Interpretation** — Uses an AI language model to interpret audio patterns (volume levels, duration, intensity) and suggest why your baby might be crying (hungry, tired, uncomfortable, needs attention, etc.)
- **Live Audio Waveform** — See a gentle, animated waveform visualization while recording so you know the app is actively listening
- **Cry History Log** — Every analysis is saved with a timestamp, detected reason, and confidence level so you can spot patterns over time
- **Parenting Insight Cards** — Each analysis result includes a brief, helpful suggestion on what to try next

## Design

- **Soft & Calming Theme** — Light pastel background with gentle lavender-to-pink gradient accents
- **Large Central Record Button** — A prominent, pulsing circle button that glows softly when actively listening
- **Animated Waveform** — Subtle, smooth audio level bars in pastel tones that dance while recording
- **Result Cards** — Rounded cards with soft shadows showing the detected cry reason, an icon, and a short tip
- **History List** — Clean timeline-style list with date grouping, each entry showing the cry reason and time
- **Dark Mode Support** — All colors adapt gracefully to dark mode using semantic system colors

## Screens

- **Home Screen** — The main recording interface with a large microphone button, live waveform visualization, and the latest analysis result displayed below
- **History Screen** — A scrollable list of all past cry analyses, grouped by date, with tap-to-expand detail for each entry
- **Tab Bar** — Two tabs: "Listen" (home) and "History"

## App Icon

- A soft pastel gradient background (lavender to light pink) with a white baby face silhouette and a small sound wave symbol — gentle and calming
